# 第16课 · MCP

## 本课目标

- 理解 MCP 让 Claude Code 连接外部系统
- 知道常见接入场景

## 16.1 MCP 是什么？

**Model Context Protocol** — 让 AI 调用外部工具的统一协议。

Claude Code 可通过 MCP 连接：

- 文档查询（Context7）
- API 文档（ApiPost）
- 数据库、内部服务
- 语雀、钉钉等

## 16.2 典型场景

| 场景 | MCP 作用 |
|------|----------|
| 写代码 | 查最新框架文档，防用过时 API |
| 协作 | 读写信雀、发钉钉消息 |
| 调试 | 查日志、调内部 API |

## 16.3 配置思路

1. 在 Claude Code 配置里加 MCP server
2. 提供认证信息（环境变量）
3. 对话中让 AI 调用对应工具
