## 自测 10 题

**Q1.** 用公司类比，进程和线程分别是什么？

<details><summary>参考答案</summary>

进程 = 独立部门（独立桌面/PID）。线程 = 部门内员工（共享部门资源）。
</details>

**Q2.** 物理内存几乎满、swap 很高，体感会怎样？为什么？

<details><summary>参考答案</summary>

卡顿。数据在桌面和仓库间频繁搬运，时间耗在 I/O 而非计算。
</details>

**Q3.** `ps aux | grep node` 里的 `|` 是什么？

<details><summary>参考答案</summary>

管道（Pipe）：前一进程标准输出接到后一进程标准输入。
</details>

**Q4.** 为什么 Mac 上 Docker 往往比 Linux 服务器更吃资源？

<details><summary>参考答案</summary>

容器依赖 Linux 内核特性；Mac 常先跑 Linux VM，再跑容器。
</details>

**Q5.** inode 和文件名是什么关系？

<details><summary>参考答案</summary>

文件名是标签；inode 是目录卡（元数据 + 数据位置）。
</details>

**Q6.** 浏览器打开 https 站点，口头说出主要步骤。

<details><summary>参考答案</summary>

DNS → TCP 握手 → TLS → HTTP 请求/响应 → 渲染。
</details>

**Q7.** SIP 和沙盒分别保护什么？

<details><summary>参考答案</summary>

SIP：系统核心少被乱改。沙盒：单个 App 只能待授权范围。
</details>

**Q8.** `write()` 返回成功，数据一定已经在磁盘上了吗？

<details><summary>参考答案</summary>

不一定，可能还在缓存；需要持久化时常靠刷盘/fsync 类机制。
</details>

**Q9.** Node 单线程为什么还能扛很多连接？

<details><summary>参考答案</summary>

事件循环 + 非阻塞 I/O：等磁盘/网络时去干别的，完成再回调。
</details>

**Q10.** 两个 AI Agent 同时改同一 git 工作区，推荐怎么隔离？

<details><summary>参考答案</summary>

git worktree（或容器）；避免多人硬改同一 working tree。
</details>
