# 第15课 · Plugins

## 本课目标

- 理解插件组成
- 知道如何选用和学习官方插件

## 15.1 插件组成

```
Plugin = Commands + Agents + Skills + Hooks
```

## 15.2 官方插件示例

| 插件 | 学什么 |
|------|--------|
| `feature-dev` | 功能开发流程 |
| `code-review` | 代码审查 |
| `commit-commands` | Git 提交 |
| `ralph-wiggum` | 迭代式自循环开发 |
| `plugin-dev` | 自己写插件 |
| `agent-sdk-dev` | Agent SDK 开发 |

路径：`code/cyc/claude-code/plugins/`

## 15.3 怎么用插件？

1. 读插件 README
2. 按说明安装/启用
3. 跑一个命令体验
4. 读源码学结构

## 本课练习

选 `feature-dev` 或 `code-review`，读 README 并列出它提供了哪些命令。

## 本课小结

> 插件是打包好的最佳实践。先会用，再学怎么写。
