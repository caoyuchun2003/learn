# 模型与成本落地手册

> **别只问「哪个模型最强」，先问「这笔账单和失败率你能不能扛」。**
>
> 面向已经会调 API、要在业务里换 DeepSeek / 文心 / 别家的人。

**作者：曹宇春**

---

## 内容简介

大模型能力卷得很快，成本结构却总被忽略：Token 计价、超时重试、多轮上下文、流式网关缓冲……都会把「测起来很便宜」变成「上线很贵」。

本书用决策表和落地清单回答：**怎么选模型、怎么控成本、怎么换模型而不炸产品**。案例贴着本站「手册助教 / 百度 CFC / 千帆」真实做法。

- 适合：Web / 全栈、做过人机对话或 RAG、要上生产的人  
- 不适合：只想背模型榜单、不碰账单与工程的人
