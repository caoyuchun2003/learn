# 操作系统白话：给开发者的第一本

> **用公司故事讲清楚电脑怎么跑**
> 零公式、对本机、能解释「为什么卡」。

**作者：曹宇春**

---

## 📕 内容简介

市面上讲操作系统的，要么是学校教材（厚、偏理论），要么是内核源码劝退。本书两者都不是——它只回答一个问题：**在 Mac（或任意电脑）上写代码的人，怎样真正听懂进程、内存、磁盘和网络。**

全书 **10 课 + 2 篇附录**，全部用「公司总管」「办公桌」「仓库」「前台工位」等类比，把 CPU、RAM、swap、文件系统、网络、容器、安全与并发翻译成大白话。每课配有 **本机只读命令**，你打开 Terminal 就能对照今天这台电脑。

适合会用 Cursor / VS Code、但没系统学过 OS 的全栈与前端开发者；也适合想给同事三分钟讲清「电脑为什么卡」的人。读完你能用类比解释卡顿、swap、Docker 吃内存，并知道该用哪几条命令自查。

---

## 👤 适合谁读

- 日常在 **Mac** 上开发，想搞懂本机卡顿原因的人
- 会写业务代码，但 **进程 / 内存 / Docker** 糊成一团的人
- 已读或并行阅读《AI 应用开发手册》，想补 **电脑底层认知** 的人
- 想用 **大白话** 给新人讲 OS，不要考试题的人
