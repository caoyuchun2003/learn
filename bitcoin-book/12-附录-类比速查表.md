# 附录 A · 类比速查表

> 教别人时随手查，一个概念一行话。
>
> **作者：曹宇春**

---

## 核心概念

| 概念 | 生活类比 | 一句话 |
|------|----------|--------|
| 比特币 | 互联网数字现金 | 规则公开、总量有限的链上资产 |
| 区块链 | 装订好的账本册子 | 一页页按顺序装订，难撕改 |
| 区块 | 账本的一页 | 约 10 分钟的交易装订在一起 |
| 交易 | 一次花钱记录 | 撕旧现金条、开新条 |
| UTXO | 未花掉的现金条 | 兜里每张钞票，不是账户余额 |
| 手续费 | 快递加急费 | 拥堵时加价才先打包 |
| 确认 | 后面又装订了几页 | 页数越多越难推翻 |
| 挖矿 | 抢盖章权 | 花电费比赛，维护账本安全 |
| 减半 | 工资每四年砍半 | 新币发行越来越慢 |
| 钱包 | 钥匙扣 | 管钥匙，不装币 |
| 助记词 | 保险柜钥匙 | 12 个词，谁要谁骗子 |
| 私钥 | 盖章的印章 | 花了钱的唯一凭证 |
| 地址 | 收款邮箱 | 可公开，别人只能往里打 |
| 交易所 | 别人开的金店 | 方便买卖，不是真拿钥匙 |
| 冷钱包 | 钥匙锁保险柜 | 大额长期更安全 |
| 热钱包 | 钥匙放口袋 | 方便，风险稍高 |
| 双花 | 同一笔钱花两次 | 公开账本要防止的问题 |
| 51% 攻击 | 买了过半彩票机 | 短期可捣乱，长期不划算 |
| 闪电网络 | 酒吧先记账后买单 | 小额频繁，最后再上主账 |
| 分叉 | 账本规则闹分裂 | 硬分叉 = 两本不同的册子 |

## 对比速查

| 维度 | 微信 / 银行 | 比特币 |
|------|-------------|--------|
| 记账 | 公司 / 银行 | 公开账本 |
| 钱 | 账户数字 | 现金条 |
| 转错 | 有时可退 | 基本不可退 |
| 丢失 | 可挂失 | 钥匙没了没了 |
| 速度 | 快 | 慢 |
| 保本 | 存款有保障 | 无，波动大 |
