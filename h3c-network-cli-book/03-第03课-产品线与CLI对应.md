# 第 03 课 · 产品线与 CLI 对应

> 什么设备，先练什么命令——别在 S5130 上找 ipsec。
>
> **作者：曹宇春**

---

## 先记住这张表

| 产品线 | 场景 | CLI 优先级（1→5） |
|--------|------|-------------------|
| **MSR810/830** | 门店 | dis int b → ip address → ip route → nat → save |
| **MSR2600** | 标准分支 | 同上 + acl + ipsec（VPN） |
| **SR6600/8800** | 核心/汇聚 | 路由基础 + bgp/mpls + dis ip rout |
| **S5130** | 接入 | vlan → port → Vlan-interface → dis vlan/mac |
| **S7500/S10500** | 核心 | S5130 技能 + irf + 三层路由 |
| **S6800/S12500** | DC | 交换 + ospf/bgp + vxlan/evpn（V7） |
| **WAC/WX** | 无线 | dis wlan ap all → dis wlan client → wlan |
| **iMC** | 网管 | Web/服务器，非 box CLI |

---

## 按网络层次

```
出口层     MSR/SR     → ip route、nat、ipsec、acl
核心层     S7500      → vlan、irf、三层路由
接入层     S5130      → vlan、access/trunk、poe
无线       WAC        → display wlan *
DC Leaf    S6800      → 接口、ospf、vxlan
DC Spine   S12500     → bgp、evpn
```

---

## 按「场景 → 命令块」

| 场景 | 命令块 |
|------|--------|
| 新交换机上架 | sysname → vlan → interface → save |
| 分支能上网 | WAN/LAN ip → 默认路由 → nat outbound |
| 分支 VPN | acl 感兴趣流 → ike → ipsec policy → apply |
| AP 上线 | 交换 PoE + vlan → AC 上 dis wlan ap all |
| DC underlay | loopback → ospf → dis ospf peer |

## 小结

- 分支 = MSR + 路由/NAT/VPN  
- 接入 = S5130 + vlan/port  
- 无线 = AC + display wlan  
- DC = OSPF/BGP + VXLAN（进阶）
