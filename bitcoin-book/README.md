# 比特币白话全书

> **零技术基础也能听懂的比特币科普**
> 用生活类比讲清楚比特币是什么、怎么用、怎么防骗——还能教给爸妈和同事。

**作者：曹宇春**

---

## 📕 内容简介

市面上讲比特币的内容，要么是炒币喊单，要么是劝退人的密码学推导。本书两者都不是——它只回答一个问题：**怎么让完全没有技术背景的普通人，真正听懂比特币是什么。**

全书 **11 课 + 2 篇附录**，全部用「现金条找零」「村里公示墙」「抢盖章装订账本」等生活类比，把 UTXO、挖矿、助记词、交易所这些概念翻译成大白话。专门配有**爸妈篇**（一页纸 + 五不要 + 误会澄清）和**同事篇**（FAQ 二十条 + 场景对照表），以及**微信短版**和**五分钟讲解术**，拿来就能发群、能打印、能当面讲。

适合想给长辈解释比特币的子女、要在同事群做科普的职场人，以及自己刚接触、想先建立正确认知的初学者。读完你能用 5 分钟讲清楚比特币，并帮身边人避开最常见的骗局和误区。
