# 第04课 · 完整公式：Model × Tools × Feedback

## 本课目标

- 在 Skill + Context 之外补全另外三个因子
- 知道日常主战场在哪

## 4.1 完整公式

```
效果 = Model × Skill × Context × Tools × Feedback
```

| 因子 | 作用 | 日常谁管 |
|------|------|----------|
| Model | 推理与生成能力 | 一般固定 |
| Skill | 会什么流程 | **你重点打磨** |
| Context | 看见什么 | **你重点装配** |
| Tools | 能否动手（bash、MCP） | Agent 自带 |
| Feedback | 如何验收 | **你要求它跑测试** |

## 4.2 为什么 Skill + Context 是主战场？

Model 通常不换；Tools Agent 有；**Skill 和 Context 每次任务都能优化**。

## 4.3 Feedback 不能省

| 坏习惯 | 好习惯 |
|--------|--------|
| 「好了」就信 | 改完跑 `pnpm test` 贴结果 |
| 不看 diff | 大改逐文件审 |
| 不跑 CI | 合并前人审 |
