# 结语

这本书想让你带走的，不是背下所有 `display`，而是一套 **可重复的实验习惯**：

> 上机先看 `display version` → 改配置进 `system-view` → 验收 `dis int b` + `dis ip routing-table` + `ping` → **`save`**。

三个 Demo 通一遍，你就从「怕 CLI」变成「能照着改 IP 交付分支和接入层」。

更深的 EVPN、IPSec 全模板、AD-NET 下发，去官方实验手册和控制器界面；日常集成与排障，本书 + 附录 A 够导航。

感谢阅读。
