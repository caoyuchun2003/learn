# 警察上门怎么办：一张纸分清情况

> 用几个具体场景，把「开不开门、跟不跟走、签不签字」说清楚。
> **一般常识说明，不是个案法律意见。**

**作者：曹宇春**

---

## 先记住一句

**有合法手续就配合；没手续可以慢慢核实；保证书大多可以不签；可以问、可以拒签，别骂、别动手。**

---

## 三种情况，别混在一起

| 他们要干嘛 | 你怎么办 |
|------------|----------|
| 随便问问、没有强制文书 | 多数情况可以 **不开门、少说话** |
| **合法传唤**（叫你去问话） | 一般要去；硬扛可能被强制到案 |
| 只让 **签保证书** | 可以不签，或看清再签 |

**证件 ≠ 想干什么都可以。**  
警察证只证明「他是警察」；传唤、搜查、拘留还要对应手续。

| 东西 | 作用 |
|------|------|
| 警察证 | 证明身份 |
| 传唤证 / 搜查证 / 拘留证等 | 证明这次有权对你做什么 |

只有证、没有对应手续 → 通常 **没有强制搜查、不能随便强制进屋**；你可以不开门、不接受搜查。

---

## 上门时就问三句

1. 请出示 **警察证**（警号、单位）。
2. **什么事？** 是传唤、搜查，还是只签材料？
3. 请出示对应文书；口头传唤请说明 **案由**。

同时：**录音**，记下警号、单位、时间。

---

## 场景一：敲门说「了解一下情况」

**他们：**「开门，我们是派出所的，了解点情况。」

**你（可以隔着门）：**

> 请出示警察证，我看一下。  
> 什么事？有传唤证或搜查证吗？

**他们：**「先开门再说。」「随便问问。」

**你：**

> 证件可以举到门镜前。  
> 没有传唤、搜查手续的话，我今天先不开门。  
> 有事请说清案由，或留下单位电话，我联系核实。

| 要点 | 说明 |
|------|------|
| 只有警察证、没强制手续 | 多数情况可不开门 |
| 态度 | 冷静核实，别骂、别拍门硬刚 |
| 留证 | 记时间、单位、警号；可录音 |

---

## 场景二：口头传唤「跟我们走一趟」

**传唤** = 依法通知你在指定时间、地点接受询问。  
**不是拘留**，但合法传唤一般应当到场。

- **书面传唤：** 出示传唤证  
- **口头传唤：** 可以没纸，但要有理由、说明案由、出示警察证，并在笔录注明口头传唤和到/离时间  

口头传唤必须有具体案件需要调查你，不是随便说一句「跟我们走」。

**他们：**「口头传唤，你必须跟我们走。」

**你：**

> 请出示警察证。  
> 口头传唤的案由是什么？我是证人还是当事人？  
> 去哪个派出所？大约多久？可否通知家属？  
> （涉及刑事嫌疑：「我需要联系律师。」）

| 情况 | 怎么办 |
|------|--------|
| 手续说清、确系合法传唤 | **应当配合**，别推搡逃跑 |
| 说不清、也不肯亮证 | 要求出示证件与案由；录音留证；可打 **12389** 或让家人核实 |

**原则：** 合法成立 → 配合；同时你可以要求说明案由、去哪、身份、通知家属，刑事相关尽快找律师。

---

## 场景三：不上门带走，只要你签保证书

**他们：**「不用走，签个保证书就行。」

**你：**

> 这是传唤，还是只签材料？  
> 请把内容给我看完整。我不签空白页。  
> 有些内容我不认可，可以不签，或注明「对第×条有异议」再签。

| 要点 | 说明 |
|------|------|
| 保证书 / 具结 / 训诫类 | 多数没有「必须签」的强制力 |
| 可以 | 不签，或看清再签 |
| 别签 | 空白页、没看懂的承认语句 |
| 先分清 | 签材料 ≠ 口头传唤必须带走 |

---

## 场景四：别用「滚」应对

| 做法 | 结果 |
|------|------|
| 核实手续、冷静提问 | ✅ |
| 骂人、动手、推搡 | ❌ 可能构成侮辱、阻碍执法，从罚款到拘留 |

被威胁时：分清是依法告知拒传唤后果，还是违法恐吓。  
留证 → **12389** / 检察院 / 律师。

---

## 一张「台词卡」（可贴备忘）

```
1. 请出示警察证（警号、单位）。
2. 请问什么事？是传唤、搜查，还是只签材料？
3. 请出示对应文书；口头传唤请说明案由。
4. 我现在录音留证。合法手续我会配合；手续不清我需要核实。
5. 保证书我需要先看完全文，不同意的部分我不签或注明异议。
6. 涉及刑事案件，我要求联系律师 / 通知家属。
```

---

## 对照表：听到这句话，像哪个场景

| 你听到的话 | 场景 | 倾向做法 |
|------------|------|----------|
| 「随便问问」「先开门」 | 场景一 | 可不开门，先看证 |
| 「口头传唤，跟我们走」 | 场景二 | 问清后，合法则配合 |
| 「签个保证就行」 | 场景三 | 可看清不签或慢签 |
| 想骂、想推 | 场景四 | 别这么干 |

---

## 当场建议流程

```
不开门（必要时）
  → 要求看警察证
  → 问：什么事？传唤还是签材料？有无文书？
  → 录音，记警号 / 单位 / 时间
  → 合法传唤：配合；只签保证书：可拒签 / 慢签
  → 刑事相关：尽快律师
  → 事后：12389 / 律师 / 保留文书复印件
```

---

## 重要声明

本文是面向普通人的 **一般法律常识整理**，不能替代律师针对具体事实的意见。  
各地程序细节可能有差异。一旦涉及传唤、搜查、拘留、刑事立案，请尽快咨询 **当地执业律师**。

投诉、监督可拨打公安机关监督电话 **12389**（以当地实际指引为准）。
