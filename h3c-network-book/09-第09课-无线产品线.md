>
> **作者：曹宇春**

---

## 先记住这个类比

| 组件 | 类比 |
|------|------|
| **WA AP** | 会议室/走廊的 **Wi-Fi 热点** |
| **AC 控制器** | **统一管所有热点的调度中心** |
| **Fit AP** | 瘦热点——自己不动脑，听 AC 的 |
| **MSR 内置 AC** | 快递站 **兼管** 门口几个热点（小分支） |
| **Cloudnet** | 用手机 **远程管连锁门店 Wi-Fi** |

```
        AC / 云管
           │
    ┌──────┼──────┐
   AP      AP      AP
    │      │      │
  S5130 PoE 交换
```

---

## 架构：集中式 AC + Fit AP（主流）

- **WA 系列 AP**：室内、室外、面板、高密  
- **WX / WAC**：独立 AC，中小 vs 大型容量  
- **MSR 内置 AC**：分支 + 少量 AP  

**Wi-Fi 代际**：Wi-Fi 5 → **6（主流）** → 6E/7（逐步部署）

---

## 选型两张表

### AP 形态

| 类型 | 场景 |
|------|------|
| 室内放装 | 办公、教室 |
| 面板 | 酒店、宿舍 |
| 室外 | 广场、厂区 |
| 高密 | 礼堂、体育馆 |

### PoE 需求

| AP | 典型 PoE |
|----|----------|
| 双频 Wi-Fi 6 | 802.3at (30W) |
| 三射频 / Wi-Fi 7 | 802.3bt (60W+) |

---

## 关键能力

- **漫游**：同 SSID 跨 AP；三层漫游要 AC 隧道  
- **802.1X / Portal**：企业 vs 访客  
- **RRM**：自动信道与功率  
- **WIDS**：无线入侵检测  

---

## 典型组网

**企业总部**

```
S7500 核心 → S5560 → S5130(PoE) → WA6320
                    WAC 380（独立 AC）
                    iMC 统一管理
```

**连锁门店**：MSR830 + Cloudnet AP

---

## 在你项目里什么时候用得上

| AP 数量 | AC 选择 |
|---------|---------|
| < 20，单分支 | MSR 内置 AC 或许够 |
| 数十～数百 | WX 系列 |
| 高校/园区上千 | WAC + 分区 |
