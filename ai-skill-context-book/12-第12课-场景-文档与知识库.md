# 第12课 · 场景作战卡：文档与知识库

## 本课目标

- 写手册、README、语雀文档的结构化协作
- 统一格式与署名

## Skill

- 固定文档结构（目标、目录、练习、小结）
- Markdown 规范
- 页脚署名、术语一致

## Context

- 现有提纲或旧稿
- 目标读者（新手 / 团队 / 客户）
- 目录结构要求
- 关联代码路径（如需引用）

## Feedback

- 人工扫一遍再发布
- 链接、代码块可渲染
- 章节编号连续

## 开口模板

```
按以下目录写第 X 课：（粘贴目录）
读者：有基础的开发者。
每课含：目标、正文、练习、小结。
```

## 与语雀配合

- 本地 md 批量整理 → 手动或脚本上传
- 文件名前缀数字控制排序（`00-`、`01-`）

## 本课练习

用模板让 AI 写一节 500 字以内的「安装说明」，你改到可发布。

## 本课小结

> 文档的 Skill 是结构模板；Context 是读者是谁、要写多深。
