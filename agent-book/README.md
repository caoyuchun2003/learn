# Agent 工程手册：Web 开发者的落地指南

> **不是 Agent 名词百科，是选型、架构与踩坑实录**
> 从「该不该上 Agent」到 MCP / Skills 实战。

**作者：曹宇春**

---

## 📕 内容简介

市面上讲 Agent 的，要么是「LLM + 工具 = Agent」一句带过，要么是框架 API 文档。本书两者都不是——它只回答一个问题：**作为要把 AI 接进产品的 Web 开发者，什么时候该用 Agent、什么时候不该，以及怎么落地才不翻车。**

全书围绕 **决策 → 模式 → 工程 → 场景** 推进：先用五个问题判断教材阅读器要不要上 Agent，再讲 Workflow vs Agent 怎么选，然后落到 RAG 答疑、工具调用、安全与成本。每课配有 **真实架构对比**（章节直灌 vs 知识库检索）和 **决策口诀**，强调能选型、能画架构，而非背 ReAct 定义。

适合已读过《AI 应用开发手册》、正在给 **网校 / 数字教材 / 内部工具** 加 AI 功能的后端与全栈开发者。

---

## 👤 适合谁读

- 产品说「我们要上 Agent」，你想判断 **是不是伪需求** 的人
- 已做 RAG 答疑，在考虑 **要不要加工具循环** 的开发者
- 读过 Agent 词条，仍不知道 **Workflow 和 Agent 怎么选** 的人
- 关心 **Key 安全、成本、幻觉、人在环** 的落地负责人
