# 第 06 课 · 三层与路由命令

> ip address、静态路由、NAT——MSR 出口三板斧。
>
> **作者：曹宇春**

---

## 1. 接口 IP

```text
[RT] interface GigabitEthernet0/0
[RT-GigabitEthernet0/0] ip address 202.1.1.2 255.255.255.252
[RT-GigabitEthernet0/0] undo shutdown
[RT-GigabitEthernet0/0] quit
```

次要地址：

```text
ip address 10.0.0.1 255.255.255.0 sub
```

撤销：`undo ip address`

---

## 2. 静态路由

```text
# 默认路由
[RT] ip route-static 0.0.0.0 0 202.1.1.1

# 明细路由（到内网网段，下一跳交换机）
[RT] ip route-static 192.168.10.0 24 192.168.0.2
```

验收：`display ip routing-table`

---

## 3. ACL

```text
[RT] acl basic 2000
[RT-acl-ipv4-basic-2000] rule 5 permit source 192.168.10.0 0.0.0.255
[RT-acl-ipv4-basic-2000] rule 10 deny source any
[RT-acl-ipv4-basic-2000] quit
```

高级 ACL（五元组）：

```text
[SW] acl advanced 3000
[SW-acl-ipv4-advanced-3000] rule 5 deny ip source 192.168.20.0 0.0.0.255 destination 192.168.10.0 0.0.0.255
[SW-acl-ipv4-advanced-3000] rule 10 permit ip
[SW-acl-ipv4-advanced-3000] quit
```

应用到接口：

```text
[SW-Vlan-interface20] packet-filter 3000 outbound
```

---

## 4. NAT 出网（MSR）

```text
[RT] acl basic 2000
[RT-acl-ipv4-basic-2000] rule 5 permit source 192.168.10.0 0.0.0.255
[RT-acl-ipv4-basic-2000] rule 10 permit source 192.168.20.0 0.0.0.255
[RT-acl-ipv4-basic-2000] quit

[RT] interface GigabitEthernet0/0
[RT-GigabitEthernet0/0] nat outbound 2000
[RT-GigabitEthernet0/0] quit
```

验收：

```text
dis nat outbound
dis nat session
ping -a 192.168.10.1 8.8.8.8
```

---

## 5. OSPF 入门（三层交换 / DC）

```text
[SW] ospf 1 router-id 1.1.1.1
[SW-ospf-1] area 0.0.0.0
[SW-ospf-1-area-0.0.0.0] network 192.168.0.0 0.0.0.255
[SW-ospf-1-area-0.0.0.0] quit
[SW-ospf-1] quit
```

验收：`display ospf peer`

---

## 6. 诊断

```text
ping 192.168.10.1
ping -a 202.1.1.2 202.1.1.1
tracert 8.8.8.8
```

---

## 小结

- 出口：ip address + ip route-static 0.0.0.0 0 + nat outbound  
- 内网回程：明细静态路由指到交换机  
- ACL 做隔离用 packet-filter
