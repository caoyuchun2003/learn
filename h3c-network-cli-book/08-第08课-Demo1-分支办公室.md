# 第 08 课 · Demo1：50 人分支办公室

> MSR2600 + S5130：双 VLAN、NAT 上网、访客隔离。
>
> **作者：曹宇春**

---

## 场景故事

分支约 50 人：**办公区 VLAN 10**、**访客 VLAN 20**，一条互联网宽带出网。  
设备：**MSR2600**（出口）+ **S5130**（接入，兼三层网关）。

---

## 拓扑

```
                    Internet
                       │
                 [ MSR2600 ]
           GE0/0 WAN      GE0/1 LAN
           202.1.1.2       192.168.0.1
                       │
                 [ S5130 ]
              GE1/0/48       GE1/0/1    GE1/0/2
            192.168.0.2      VLAN10     VLAN20+PoE
                              │           │
                            员工 PC      访客/AP
```

---

## IP 规划

| 区域 | 网段 | 网关 |
|------|------|------|
| MSR↔SW 互联 | 192.168.0.0/24 | MSR .1 / SW .2 |
| 办公 VLAN10 | 192.168.10.0/24 | 192.168.10.1（SW） |
| 访客 VLAN20 | 192.168.20.0/24 | 192.168.20.1（SW） |
| WAN | 202.1.1.0/30 | MSR .2，ISP .1 |

---

## 一、MSR2600 配置

```text
<H3C> system-view
[H3C] sysname Branch-MSR

# WAN
[Branch-MSR] interface GigabitEthernet0/0
[Branch-MSR-GigabitEthernet0/0] description WAN-Internet
[Branch-MSR-GigabitEthernet0/0] ip address 202.1.1.2 255.255.255.252
[Branch-MSR-GigabitEthernet0/0] undo shutdown
[Branch-MSR-GigabitEthernet0/0] quit

# LAN
[Branch-MSR] interface GigabitEthernet0/1
[Branch-MSR-GigabitEthernet0/1] description LAN-to-Switch
[Branch-MSR-GigabitEthernet0/1] ip address 192.168.0.1 255.255.255.0
[Branch-MSR-GigabitEthernet0/1] undo shutdown
[Branch-MSR-GigabitEthernet0/1] quit

# 路由
[Branch-MSR] ip route-static 0.0.0.0 0 202.1.1.1
[Branch-MSR] ip route-static 192.168.10.0 24 192.168.0.2
[Branch-MSR] ip route-static 192.168.20.0 24 192.168.0.2

# NAT
[Branch-MSR] acl basic 2000
[Branch-MSR-acl-ipv4-basic-2000] rule 5 permit source 192.168.10.0 0.0.0.255
[Branch-MSR-acl-ipv4-basic-2000] rule 10 permit source 192.168.20.0 0.0.0.255
[Branch-MSR-acl-ipv4-basic-2000] quit
[Branch-MSR] interface GigabitEthernet0/0
[Branch-MSR-GigabitEthernet0/0] nat outbound 2000
[Branch-MSR-GigabitEthernet0/0] quit

[Branch-MSR] save
```

---

## 二、S5130 配置

```text
<H3C> system-view
[H3C] sysname Branch-SW

# VLAN
[Branch-SW] vlan 10
[Branch-SW-vlan10] name Office
[Branch-SW-vlan10] quit
[Branch-SW] vlan 20
[Branch-SW-vlan20] name Guest
[Branch-SW-vlan20] quit

# 上联 MSR（三层路由口）
[Branch-SW] interface GigabitEthernet1/0/48
[Branch-SW-GigabitEthernet1/0/48] description Uplink-to-MSR
[Branch-SW-GigabitEthernet1/0/48] port link-mode route
[Branch-SW-GigabitEthernet1/0/48] ip address 192.168.0.2 255.255.255.0
[Branch-SW-GigabitEthernet1/0/48] undo shutdown
[Branch-SW-GigabitEthernet1/0/48] quit

# 办公接入
[Branch-SW] interface GigabitEthernet1/0/1
[Branch-SW-GigabitEthernet1/0/1] port link-type access
[Branch-SW-GigabitEthernet1/0/1] port access vlan 10
[Branch-SW-GigabitEthernet1/0/1] undo shutdown
[Branch-SW-GigabitEthernet1/0/1] quit

# 访客/AP
[Branch-SW] interface GigabitEthernet1/0/2
[Branch-SW-GigabitEthernet1/0/2] port link-type access
[Branch-SW-GigabitEthernet1/0/2] port access vlan 20
[Branch-SW-GigabitEthernet1/0/2] poe enable
[Branch-SW-GigabitEthernet1/0/2] undo shutdown
[Branch-SW-GigabitEthernet1/0/2] quit

# 网关 SVI
[Branch-SW] interface Vlan-interface 10
[Branch-SW-Vlan-interface10] ip address 192.168.10.1 255.255.255.0
[Branch-SW-Vlan-interface10] quit
[Branch-SW] interface Vlan-interface 20
[Branch-SW-Vlan-interface20] ip address 192.168.20.1 255.255.255.0
[Branch-SW-Vlan-interface20] quit

# 默认路由
[Branch-SW] ip route-static 0.0.0.0 0 192.168.0.1

# 访客不能访问办公（可选）
[Branch-SW] acl advanced 3000
[Branch-SW-acl-ipv4-advanced-3000] rule 5 deny ip source 192.168.20.0 0.0.0.255 destination 192.168.10.0 0.0.0.255
[Branch-SW-acl-ipv4-advanced-3000] rule 10 permit ip
[Branch-SW-acl-ipv4-advanced-3000] quit
[Branch-SW] interface Vlan-interface 20
[Branch-SW-Vlan-interface20] packet-filter 3000 outbound
[Branch-SW-Vlan-interface20] quit

[Branch-SW] save
```

## 三、PC 参数

**办公 PC**（接 Gi1/0/1）：IP `192.168.10.10/24`，网关 `192.168.10.1`，DNS `8.8.8.8`

**访客 PC**（接 Gi1/0/2）：IP `192.168.20.10/24`，网关 `192.168.20.1`

---

## 四、验收 checklist

### MSR

```text
□ dis int b                    → GE0/0、GE0/1 UP
□ dis ip routing-table         → 0.0.0.0/0 → 202.1.1.1
□ dis nat outbound             → GE0/0 绑定 acl 2000
□ ping 202.1.1.1
□ ping 192.168.0.2
□ ping 192.168.10.1
```

### S5130

```text
□ dis vlan                     → 10、20 与端口对应
□ dis ip routing-table         → 0.0.0.0/0 → 192.168.0.1
□ ping 192.168.0.1
```

### PC

```text
□ ping 192.168.10.1            → 网关通
□ ping 8.8.8.8                 → 上网通
□ 访客 ping 192.168.10.10      → 应被 ACL deny（若已配隔离）
```

---

## 五、包怎么走（办公 PC ping 8.8.8.8）

```
PC → 192.168.10.1(SW SVI) → 0.0.0.0/0 → MSR 192.168.0.1
→ NAT 为 202.1.1.2 → ISP → Internet
```

## 六、翻车排查

| 现象 | 查 |
|------|-----|
| PC ping 不通网关 | dis vlan、口是否 access 对口、PC 网关 |
| SW ping 不通 MSR | 互联口 IP、port link-mode route |
| 网关通上不了网 | MSR 默认路由、nat outbound、acl 网段 |
| 重启配置没了 | 忘了 save |
