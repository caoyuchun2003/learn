# 配合执法时的六句沟通话术

> 依法配合、冷静核实。紧张时可以照着说。
> **一般常识说明，不是个案法律意见。**

**作者：曹宇春**

---

## 六句话（可直接说）

```
1. 请出示警察证（警号、单位）。
2. 请问什么事？是传唤、搜查，还是只签材料？
3. 请出示对应文书；口头传唤请说明案由。
4. 我现在录音留证。合法手续我会配合；手续不清我需要核实。
5. 保证书我需要先看完全文，不同意的部分我不签或注明异议。
6. 涉及刑事案件，我要求联系律师 / 通知家属。
```

---

## 每一句在干什么

| 句 | 目的 |
|----|------|
| **1** | 依法核实执法身份，记下警号、单位 |
| **2** | 分清事由：询问、传唤、搜查，还是签署材料 |
| **3** | 了解法律文书；口头传唤应说明案由 |
| **4** | 表明配合合法执法，同时对手续留存核实记录 |
| **5** | 仔细阅读后再签署；对有异议内容可注明或暂缓签署 |
| **6** | 涉及刑事案件时，依法联系律师并通知家属 |

---

## 怎么用

- 存手机备忘录，敲门时打开对照。
- 语气平静、态度配合；先问清身份与事由。
- **依法出示证件与手续后，应予配合；不要谩骂、不要动手。**
- 更完整的情景说明，见：[执法敲门时怎么沟通](执法敲门时怎么沟通.md)

---

## 极简版（三句）

```
请出示警察证。
请问什么事？传唤、搜查，还是签署材料？请出示文书或说明案由。
我依法配合并留存记录；材料需看清再签；刑事案件请允许联系律师。
```
