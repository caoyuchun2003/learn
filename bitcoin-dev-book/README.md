# 比特币技术全书

> **开发者视角的比特币协议学习手册**
> 从密码学、UTXO、Script 到节点运维与交易构造，系统搞懂 BTC 技术栈。

**作者：曹宇春**

---

## 📕 内容简介

市面上讲比特币的内容，要么是炒币叙事，要么是碎片化博客。本书面向**有编程基础的开发者**，从协议第一性原理出发，系统讲清比特币如何工作、如何验证、如何开发。

全书 **21 课 + 3 篇附录**，覆盖密码学基础、UTXO 与交易结构、Bitcoin Script、区块与 PoW 共识、SegWit/Taproot、P2P 网络与全节点、Bitcoin Core 实战、PSBT 多签、安全威胁与闪电网络入门。每课配有**技术要点、数据结构、命令示例**，强调能读白皮书、能跑 regtest、能构造签名交易。

适合想深入理解比特币协议的后端 / 全栈开发者、区块链工程师，以及读完《比特币白话全书》想继续下钻的读者。

---

## 👤 适合谁读

- 有 **编程基础**（至少一门语言 + 基本命令行）
- 想搞懂 **UTXO、Script、PoW** 而非只会调 API 的开发者
- 需要 **跑节点、构造交易、理解软分叉** 的工程师
- 已读白话版、想读 **白皮书 + 源码** 的进阶者

## 🧭 怎么读

- **只想建立技术全景** → 读前言 + 第 01、07、08 课
- **系统学习** → 按 01 → 21 顺序，配合附录 C 的 10 周计划
- **做开发** → 重点读第 15–18 课，动手跑 regtest
- **查某一主题** → 直接跳对应课（如第 12 课 Taproot）
