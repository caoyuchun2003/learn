# 华三网络基础：从 Comware 到云网一体

> **用园区城市类比讲清楚路由、交换、无线与网管**
> 面向政企组网选型，能解释「这套设备是干什么的」。

**作者：曹宇春**

---

## 📕 内容简介

市面上讲华三/新华三的，要么是官方配置手册（厚、偏命令），要么是零散产品彩页。本书两者都不是——它只回答一个问题：**做园区网、分支网或数据中心组网时，华三「网络基础」产品线各系列是干什么的、怎么搭配。**

全书 **10 课 + 4 篇附录**，用「园区城市、快递站、楼层交换、Wi-Fi 覆盖、交通指挥中心」等类比，把 **MSR/SR 路由、S 系列交换、WA 无线、iMC/AD-NET 网管** 翻译成大白话。每课配有 **选型对照表与组网图**，你看完方案书能对着型号问对问题。

适合做 **网络售前、集成、运维** 的工程师；也适合开发者和 IT 负责人，想搞懂机房和园区里那些盒子分工的人。

---

## 👤 适合谁读

- 接触华三/新华三组网方案，但 **型号太多理不清** 的人
- 会配 VLAN，但 **MSR 和 SR、S5130 和 S12500 分不清场景** 的人
- 需要给同事 **三分钟讲清一套园区网怎么搭** 的人
- 想系统了解 **Comware、IRF、VXLAN、SD-WAN** 在华三产品里对应什么的人
