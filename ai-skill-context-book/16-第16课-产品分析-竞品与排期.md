# 第16课 · 产品分析：竞品与排期

## 本课目标

- 竞品对比有维度、有结论
- 功能排期有依据

## 16.1 竞品分析 Skill

输出 **对比表** + **三类建议**：

| 维度 | 竞品A | 竞品B | 我们 |
|------|-------|-------|------|
| 目标用户 | | | |
| 核心场景 | | | |
| 核心功能 | | | |
| 定价 | | | |
| 优劣势 | | | |

结尾：
- 建议 **跟进** 的功能 3 条
- 建议 **差异化** 的 3 条
- 建议 **放弃跟进** 的 3 条

无依据标「待核实」。

## 16.2 Context

- 我们产品一句话 + 链接
- 竞品名单（3–5 个）+ 你知道的功能点
- 分析目的：定价 / 差异化 / 功能取舍

## 开口模板 B：竞品

```
【我们的产品】…
【竞品】A / B / C（附链接或功能点）
【目的】定差异化

输出对比表 + 跟进/差异化/放弃 各 3 条建议。
```

## 16.3 排期 Skill：RICE / ICE

| 方法 | 适合 |
|------|------|
| RICE | Reach × Impact × Confidence / Effort |
| ICE | Impact × Confidence × Ease |

## 16.4 Context（排期）

- 候选功能列表
- 约束：时间、人力、必须私有化等
- 用户反馈、战略优先级

## 开口模板 C：排期

```
【背景】2-3 句
【候选功能】1… 2… 3…
【约束】2 周 / 1 人 / 内网部署
用 RICE 排序，给出 TOP2、可延后、明确不做。
```

## 本课练习

选 2 个竞品填对比表（可简写），写 1 条差异化建议。

## 本课小结

> 竞品分析的价值是「往哪切入」，排期的价值是「先不做什么」。
