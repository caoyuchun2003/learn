# 第02课 · 安装与环境

## 本课目标

- 完成安装并验证
- 能在项目目录启动 Claude Code

## 2.1 环境要求

- Node.js 18+（部分安装方式需要）
- macOS / Linux / Windows
- Anthropic 账号

## 2.2 安装（推荐）

**macOS / Linux：**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Homebrew：**

```bash
brew install --cask claude-code
```

**验证：**

```bash
claude --version
```

> npm 全局安装已 deprecated，优先用上面两种方式。

## 2.3 第一次启动

```bash
cd ~/your-project
claude
```

首次会引导登录 Anthropic 账号。

## 2.4 常用命令

| 场景 | 命令 |
|------|------|
| 正常进入 | `claude` |
| 继续上次 | `claude --continue` |

## 本课练习

1. 安装并运行 `claude --version`
2. 在 `learn` 目录启动，问：「这个目录有什么？先别改文件。」

## 本课小结

> 养成「先进项目目录再 claude」的习惯。
