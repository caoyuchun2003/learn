# 第 03 课 · MSR 分支路由器

> 华三面向分支的「多业务盒子」：VPN、双 WAN、可选无线 AC。
>
> **作者：曹宇春**

---

## 先记住这个类比

**MSR（Multi-Service Router）** = 每个分支的 **快递总站 + 门卫**：

- **收发货** → 路由、NAT、多 WAN  
- **加密专线** → IPSec/SSL VPN 连总部  
- **安检** → 内置防火墙、ACL  
- **顺便管 Wi-Fi**（部分型号）→ 内置 AC，小分支不用单独买控制器  

```
互联网 / 专线 / 4G
        │
   ┌────┴────┐
   │   MSR   │
   └────┬────┘
        │
   S5130 + AP
```

---

## 型号梯度（选型第一眼）

| 系列 | 场景 | 一句话 |
|------|------|--------|
| MSR810/830/930 | 门店、小微 | 桌面、4G、基础 VPN |
| **MSR2600** | **标准分支** | **政企销量主力，方案模板** |
| MSR3600 | 中大型分支 | 更高 VPN 吞吐、多 WAN |
| MSR5600 | 大分支/双机 | 多槽位、冗余 |

```
性能 ↑  810 → 2600 → 3600 → 5600
```

---

## 核心能力

| 能力 | 说明 |
|------|------|
| 多 WAN | 专线 + 互联网 + 4G/5G 备份，链路探测切换 |
| IPSec VPN | 分支连总部 Hub，政企标配 |
| SSL VPN | 移动办公远程接入 |
| 安全 | NAT、状态防火墙、防攻击 |
| 内置 AC | 少量 AP 场景，极简分支 |

---

## 标准分支模板

```
              ┌── 总部 SR/MSR（VPN Hub）──┐
              └───────────▲──────────────┘
                          │ IPSec
              ┌───────────┴──────────────┐
              │       分支 MSR2600        │
              │  WAN1 专线  WAN2 4G 备    │
              └───────────┬──────────────┘
                          │
              ┌───────────┴──────────────┐
              │  S5130 PoE + WA6320 AP   │
              └──────────────────────────┘
```

---

## AD-WAN：MSR 的进化方向

传统：逐台配 IPSec，链路坏手工切。  
**AD-WAN**：控制器集中下发，**应用级选路**（视频会议走专线、网页走宽带），**ZTP** 零接触开局。

MSR today 常作为 **SD-WAN CPE**，不只是「盒子路由器」（第 10 课）。

## 在你项目里什么时候用得上

| 需求 | 选什么 |
|------|--------|
| 50 人分支 + VPN | MSR2600 |
| 连锁单店 + 4G | MSR810/830 |
| 200+ 人 + 双机 | MSR5600 |
| 多链路智能选路 | MSR + AD-WAN |

## 小结

- MSR = 分支出口 + VPN + 安全 + 可选无线 AC  
- 2600 是标准档位；810/5600 是两极  
- AD-WAN 让 MSR 成为 SD-WAN 边缘节点
