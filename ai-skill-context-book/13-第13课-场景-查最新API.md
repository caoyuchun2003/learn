# 第13课 · 场景作战卡：查最新 API

## 本课目标

- 防止 AI 用过时 API
- 先查文档再写代码

## Skill

- **不信训练记忆**，以官方文档为准
- 先查后写
- 注明版本号

## Context

- Context7 / 官方文档 MCP 查询结果
- `package.json` 或 lockfile 里的版本
- 框架主版本（Next 15 vs 14）

## Feedback

- 代码中的 API 与文档一致
- 能说出依据哪版文档

## 开口模板

```
我要用 Next.js App Router 做服务端 action。
先查最新文档，再给可运行示例。
项目版本：Next 15.x。不要用过时 Pages Router 写法。
```

## 无 MCP 时

- 贴官方文档链接或段落
- 或让 AI 列出「我不确定是否过时的点」你再核实

## 本课练习

让一个你不太熟的库写示例，强制它先声明查了什么文档。

## 本课小结

> 查 API 的 Context 是版本 + 文档原文，不是「我记得是这样」。
