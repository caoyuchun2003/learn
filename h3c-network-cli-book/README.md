# 华三网络基础 CLI 实操：产品线、命令与场景 Demo

> **从 Comware 视图到分支/园区/DC 三套可验收实验**
> 能配、能验、能排障。

**作者：曹宇春**

---

## 📕 内容简介

已有《华三网络基础》讲清 **MSR、S 系列、无线、网管是什么**；本书解决 **怎么在 Comware CLI 上配出来**。

全书 **12 课 + 4 篇附录**，按 **CLI 基础 → 命令分类手册 → 三个完整 Demo → 排障与自动化** 推进。含 **50 人分支（MSR+S5130）、园区 Trunk 接入、DC Leaf 入门** 三套可复制命令与验收清单。

适合 **运维、集成、实验学习者**；也适合读完产品线书后想动手的人。

---

## 👤 适合谁读

- 知道华三型号，但 **不敢上 CLI** 的人
- 会 `ping`，但 **`display` / `undo` / `save` 分不清** 的人
- 需要 **照着 Demo 在实验台验收** 的人
- 想查 **命令分类速查表** 的人

| 你的需求 | 读哪里 |
