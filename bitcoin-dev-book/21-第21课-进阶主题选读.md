## Layer 1 扩展与数据

| 主题 | 说明 |
|------|------|
| **Ordinals / Inscriptions** |  witness 中嵌入数据，NFT 争议 |
| **Runes / BRC-20** | 基于 UTXO 的同质化代币实验 |
| **Drivechain** | 侧链 BIP 提案（未激活） |
| **CTV / APO** |  covenant 类软分叉提案讨论 |

了解即可；**协议主干仍是 P2P 现金 + UTXO**。

---

## 隐私技术

| 技术 | 原理 |
|------|------|
| CoinJoin | 多用户合并输入/output 模糊关联 |
| PayJoin (P2EP) | 付款同时接收，打破启发式 |
| Silent Payments (BIP-352) | 静态地址，链上不可关联 |
| Dandelion++ | 交易传播隐私 |

---

## 侧链与联邦链

| 项目 | 模型 |
|------|------|
| Liquid | Blockstream 联邦侧链，机密资产 |
| Rootstock (RSK) | 合并挖矿 EVM 侧链 |
| Fedimint | 社区联邦 Chaumian mint |

与主链**双向锚定**或**信任联邦**——安全模型不同。

## 挖矿与协议研究

- Stratum v2：去中心化矿池协议
- BetterHash：矿工自选区块模板
- **UTXO commitments**：轻客户端验证 UTXO 集

## 读源码入门

Bitcoin Core 推荐阅读顺序：

```
src/consensus/     共识常量
src/script/        脚本解释器
src/validation.cpp 区块/交易验证主逻辑
src/net_processing.cpp  P2P
src/wallet/        钱包（若关心应用）
```

配合 [bitcoinbook](https://github.com/bitcoinbook/bitcoinbook) 第三版（《精通比特币》）。

## 与以太坊开发衔接

若你同时学 ETH：

- BTC 精通 **UTXO、交易、节点**
- ETH 精通 **账户、合约、Gas**
- 跨链桥是**高信任/高风险**组件，单独审计

## 本课小结

- 比特币生态在 L1 保守演进（软分叉为主）
- 实验多在二层、侧链、客户端外协议
- 深入方向：闪电、隐私、源码、挖矿经济
