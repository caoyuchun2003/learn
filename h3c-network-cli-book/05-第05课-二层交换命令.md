# 第 05 课 · 二层交换命令

> vlan、access、trunk、Vlan-interface——S5130 的日常。
>
> **作者：曹宇春**

---

## 先记住

| 概念 | 命令关键词 |
|------|------------|
| 划部门 | `vlan` |
| 接电脑 | `port link-type access` |
| 上联传多 VLAN | `port link-type trunk` |
| 三层网关 | `interface Vlan-interface` |
| 供电 | `poe enable` |

---

## 1. 创建 VLAN

```text
[SW] vlan 10
[SW-vlan10] name Office
[SW-vlan10] quit

[SW] vlan 20
[SW-vlan20] name Guest
[SW-vlan20] quit
```

撤销：`undo vlan 20`

---

## 2. Access 口（接 PC / AP）

```text
[SW] interface GigabitEthernet1/0/1
[SW-GigabitEthernet1/0/1] port link-type access
[SW-GigabitEthernet1/0/1] port access vlan 10
[SW-GigabitEthernet1/0/1] undo shutdown
[SW-GigabitEthernet1/0/1] quit
```

---

## 3. Trunk 口（上联核心）

```text
[SW] interface Ten-GigabitEthernet1/0/49
[SW-Ten-GigabitEthernet1/0/49] port link-type trunk
[SW-Ten-GigabitEthernet1/0/49] port trunk permit vlan 10 20
[SW-Ten-GigabitEthernet1/0/49] undo shutdown
[SW-Ten-GigabitEthernet1/0/49] quit
```

---

## 4. 三层网关 SVI

```text
[SW] interface Vlan-interface 10
[SW-Vlan-interface10] ip address 192.168.10.1 255.255.255.0
[SW-Vlan-interface10] quit
```

---

## 5. PoE（AP / 摄像头）

```text
[SW-GigabitEthernet1/0/2] poe enable
```

验收：`display poe interface`

---

## 6. 路由口（与 MSR 互联，Demo 常用）

```text
[SW] interface GigabitEthernet1/0/48
[SW-GigabitEthernet1/0/48] port link-mode route
[SW-GigabitEthernet1/0/48] ip address 192.168.0.2 255.255.255.0
[SW-GigabitEthernet1/0/48] undo shutdown
[SW-GigabitEthernet1/0/48] quit
```

> 部分版本用 `port link-type route`，以 `?` 和官方手册为准。

---

## 7. 链路聚合（概念）

```text
[SW] interface Bridge-Aggregation1
[SW-Bridge-Aggregation1] quit
[SW] interface GigabitEthernet1/0/47
[SW-GigabitEthernet1/0/47] port link-aggregation group 1
```

---

## 常用 undo

| 原配置 | 撤销 |
|--------|------|
| `port access vlan 10` | `undo port access vlan` |
| `poe enable` | `undo poe enable` |
| `shutdown` | `undo shutdown` |

---

## 练习

1. 写命令：VLAN 30 名 Server，Gi1/0/5 做 access vlan 30。  
2. Trunk 口如果不 permit 某 VLAN，会怎样？  

---

## 小结

- 接入 access + vlan；上联 trunk + permit vlan  
- 网关用 Vlan-interface + ip address  
- 连路由器可用 port link-mode route
