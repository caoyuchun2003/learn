## 自测题

1. 用户视图和系统视图提示符分别是什么？怎么进系统视图？  
2. 改完配置为什么要 `save`？  
3. 巡检三板斧是哪三条 display？  
4. `undo shutdown` 相对 `shutdown` 是什么关系？  
5. access 口和 trunk 口各连什么？  
6. 分支上网需要 MSR 上哪三类配置（概念）？  
7. NAT 通常绑在 WAN 还是 LAN 口？  
8. 网关配在 Vlan-interface 表示什么？  
9. Trunk 上 `port trunk permit vlan` 漏了某 VLAN 会怎样？  
10. `dis wlan ap all` 里 AP 状态 Run 表示什么？  
11. VPN 验收先 dis 哪两个？  
12. DC Underlay 为什么用 Loopback + OSPF？  
13. 口 Up 但 ping 不通网关，先查 vlan 还是 nat？  
14. 100 台设备批量改 VLAN，CLI 还是 NETCONF 更合适？  
15. Demo1 中，PC ping 8.8.8.8 经过 MSR 的哪个功能转换源 IP？  

---

## 参考答案要点

1. `<>` / `[]`；`system-view`  
2. 不 save 重启丢运行配置  
3. dis int b、dis ip routing-table、dis cur  
4. undo = 反向/删除  
5. access 接终端；trunk 上联传多 VLAN  
6. ip address、默认路由、nat outbound（+ acl）  
7. WAN  
8. 该 VLAN 的三层网关 IP  
9. 该 VLAN 跨 Trunk 不通  
10. AP 正常工作在线  
11. dis ike sa、dis ipsec sa  
12. 稳定 router-id，Spine-Leaf 可达  
13. 先 vlan（二层），不是 nat  
14. NETCONF/Ansible  
15. NAT
