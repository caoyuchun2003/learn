# 第01课 · AI 使用的核心：Skill 与 Context

## 本课目标

- 建立「Skill + Context」双核心思维
- 告别「背 prompt 咒语」

## 1.1 核心公式

```
AI 使用效果 ≈ Skill（会什么）× Context（此刻看见什么）
```

- **Skill**：可被调用的能力模块——流程、规范、领域知识、工具用法
- **Context**：当次进入模型的一切信息——文件、报错、历史、CLAUDE.md、检索结果

## 1.2 厨师类比

| | 对应 |
|---|------|
| 菜谱 | Skill |
| 食材 | Context |
| 成品 | AI 输出 |

菜谱再好，食材不对做不出来；食材齐全没菜谱会瞎做。

## 1.3 每次任务前过三问

```
□ Skill：这类事的标准流程是什么？
□ Context：它必须看见哪些信息？
□ Feedback：怎么证明做完了？
```

## 1课练习

拿今天一个真实任务，写下它的 Skill 和 Context 各 3 条。

## 本课小结

> 用 AI 的本质：打磨 Skill，装配 Context。Prompt 只是 Context 的入口。
