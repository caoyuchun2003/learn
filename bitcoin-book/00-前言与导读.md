## 📌 导读（全景总结）

### 一切的地基：三句话

> 全书长在这三个根上，所有「为什么」都能追到这里。

```
① 比特币 = 互联网上的数字现金 + 全世界都能看的公开账本
② 助记词（12 个词）= 保险柜钥匙，谁拿到谁就能动你的钱
③ 花钱像拿钞票：给对方、找零给自己，转错了基本退不了
```

### 全书分四篇

```
入门篇：搞懂是什么（第 01–04 课）
爸妈篇：给长辈的定心丸（第 05–07 课）
同事篇：职场人的理性参考（第 08–09 课）
你来教：拿来就能用（第 10–11 课）
```

### 三个核心比喻（全书反复用）

| 技术概念 | 生活类比 |
|----------|----------|
| 比特币本身 | 互联网上的数字现金 |
| 公开账本 | 村里公示墙，大家都能看、都能核对 |
| 花钱（UTXO） | 拿钞票付款，有找零，不是改银行卡数字 |
| 助记词 | 保险柜钥匙 |
| 挖矿 | 抢盖章权，每 10 分钟装订一页账本 |
| 交易所 | 别人开的金店，方便买卖但不是真拿钥匙 |

### 五条核心心法（只记这 5 句）

1. **搞懂不必，安全要懂**——管好 12 个词，比搞懂挖矿重要一万倍
2. **钥匙不在你手里，币就不算你的**——放交易所 = 钱在别人金店
3. **像现金，不像银行卡**——转错了基本退不了，丢了钥匙基本找不回
4. **了解可以，冲动不行**——别拿养老钱、看病钱去赌
5. **防的是骗子，不是技术**——机制公开，骗局在人和项目

## 电梯演讲（30 秒版）

> 比特币是一种放在互联网上的数字现金，账本是全世界都能看的。您的 12 个词就是保险柜钥匙，丢了找不回来。花钱像拿钞票：给对方、找零给自己，转错了基本退不了。方便不如微信，特点是规则公开、总量有限。最重要：别信陌生人，别拿养老钱去赌。

---

## 讲解顺序（通用，5 分钟）

1. **数字现金 + 公开账本**（2 分钟）
2. **助记词 = 保险柜钥匙**（1 分钟）
3. **五不要 / 安全提醒**（2 分钟）

**千万别讲：** 哈希、算法、区块高度、UTXO 英文缩写——普通人听了只会更晕。
