## 📌 导读（全景总结）

### 一切的地基：三句话

```
① 视图：<> 只能看，[] 才能改——先 system-view 再动手
② 三板斧：display 看、undo 删、save 存——不 save 重启全没
③ 按场景学命令：分支玩路由+NAT，接入玩 vlan+port，DC 玩路由+overlay
```

### 全书分四篇

```
CLI 基础：视图、巡检、产品线对应（第 01–03 课）
命令分类：display / 二层 / 三层 / VPN无线（第 04–07 课）
场景 Demo：分支、园区、DC 三套实验（第 08–10 课）
排障自动化：现象→命令、NETCONF 入门（第 11–12 课）
```

### CLI 与产品线对应（核心表）

| 产品线 | CLI 最先学 |
|--------|------------|
| MSR2600 | ip address、ip route-static、nat、acl |
| S5130 | vlan、port link-type、Vlan-interface |
| S7500 | 同上 + irf、三层路由 |
| S6800/S12500 | 交换基础 + ospf/bgp + vxlan（V7） |
| WAC | display wlan ap all、wlan 视图 |
| iMC | Web/API，不是 box CLI |

### 五条 CLI 心法

1. **改前先 dis cur 备份**——记事本存一份  
2. **改上联口要有 Console**——别把自己关外面  
3. **验收固定顺序**：dis int b → dis ip routing-table → ping  
4. **版本先看 display version**——V5/V7 语法有差异  
5. **Demo 通一遍再改生产**——参数可以错，流程不能乱

## 电梯演讲（30 秒版）

> 华三网络设备都跑 Comware：`sys` 进配置，`display` 看，`undo` 删，完事 `save`。分支 Demo 是 MSR 做 NAT 出网、S5130 划 VLAN 当网关；园区 Demo 练 Trunk；DC Demo 练 Loopback+OSPF。附录有命令速查表。
