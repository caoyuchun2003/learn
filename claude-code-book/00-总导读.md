## 📕 内容简介

Claude Code 是跑在终端里的 AI 编程 Agent，能读代码、改文件、跑命令、处理 Git。市面上讲 AI 编程工具的多是零散技巧，本书则提供一条**从安装到插件开发的完整路径**。

全书 **20 课 + 3 篇附录**，按「会用 → 会聊 → 会配 → 会搭 → 会造」五个阶段推进：从第一次对话、提示工程与上下文管理，到 `CLAUDE.md`、Git 工作流与 Code Review，再到 Skills、Hooks、Plugins、MCP，最后落到团队规范与自建插件。每课附有练习与小结，强调**能动手、能验收**，而非背命令。

适合已会用 ChatGPT 或 Cursor、想把 **Claude Code 当成日常开发主力工具** 的开发者。读完你能独立安装配置、写出清晰任务、扩展插件与 MCP，并在团队里建立可复用的协作习惯。

## 👤 适合谁读

- 想用 **终端 Agent** 提升开发效率的后端 / 全栈开发者
- 已用 Cursor 等工具，想补 **Claude Code** 这一块的开发者
- 需要把 AI 协作流程 **标准化到团队** 的技术负责人
- 想搞懂 Skills、Hooks、Plugins、MCP 是什么的人

## 🧭 怎么读

- **只想快速上手** → 读第 01–03 课，完成第一个小任务
- **系统学习** → 按 01 → 20 顺序，配合附录 30 天计划
- **查某一主题** → 直接跳对应课（如第 16 课 MCP）
- **和协作方法论配合** → 先读《AI 协作方法论》再读本课，事半功倍

## 核心心法

> 把 AI 当「聪明但对你项目一无所知的新同事」——任务说清、上下文给足、改完要验证。

## 每天学习循环

1. **读** 一节（15 分钟）
2. **做** 课后练习（30 分钟）
3. **记** 一句心得
4. **写** 进笔记（费曼学习法）
