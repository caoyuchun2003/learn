# 第12课 · Slash Commands

## 本课目标

- 了解内置斜杠命令
- 知道如何固化常用流程

## 12.1 内置命令

| 命令 | 作用 |
|------|------|
| `/help` | 查看帮助 |
| `/bug` | 报告问题 |

具体命令以你安装的版本为准，运行 `/help` 查看完整列表。

## 12.2 自定义命令的价值

把团队重复用的 prompt 固化成命令，例如：

- `/review` — 审查当前 diff
- `/explain` — 解释当前文件结构
- `/onboard` — 新人环境检查

## 12.3 设计原则

1. **一个命令一件事**
2. **参数尽量少**
3. **输出格式固定**（方便脚本接）

## 本课练习

定义一个你常用的命令需求（写下来即可），思考它应接受什么参数。

## 本课小结

> Slash Commands = 把经验变成一键复用的按钮。
