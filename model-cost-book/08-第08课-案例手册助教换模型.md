# 第 08 课 · 案例：手册助教换模型

> 结论：**换模型 = 改配置 + 回归黄金问题 + 看页脚 model 字段；不是重写半个产品。**

---

## 背景

learn 手册助教：

- 前端：VitePress「问手册」  
- 后端：百度 CFC `/api/ask`、`/api/ask/stream`  
- 模型：经千帆；默认曾用 `ernie-x1.1`，后改为 `deepseek-v3.2`

## 实际步骤（可复用）

1. **用当前 Key 拉模型列表 / 逐个试调**，确认目标模型可调用  
2. 改 `functions/template.yaml` 与代码默认：`QIANFAN_MODEL`  
3. `export QIANFAN_AK=... && ./scripts/deploy.sh`  
4. `curl` 看响应里 `"model":"deepseek-v3.2"`  
5. 用 3～5 道黄金问题点「问手册」人工验收  
6. 确认页脚 `mode: qianfan`、`stream: fake|sse`

## 换模时观察什么

| 观察 | 含义 |
|------|------|
| `invalid_model` | 账号未开通或名字写错 |
| 拒答变多 | 提示过严 + 风格变化，调 system |
| 变慢 | 换 flash 档或升超时 |
| 相关阅读在、正文空 | 流式/解析路径问题，不是「没检索」 |
